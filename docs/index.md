## Get started

Pick a language:

- **[Python](python.md)**: `pip install scopemask`
- **[JavaScript / TypeScript](js.md)**: `npm install scopemask`
- **[Go](golang.md)**: `go get github.com/khan-asfi-reza/scopemask/go@latest`

Built on [sqids](https://sqids.org) for the short, reversible encoding.
